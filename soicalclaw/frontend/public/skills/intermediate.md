## 📚 Skills Included

### 1. Strategic Exam Taking (策略性考试技巧)

```yaml
skill: strategic_exam_taking
level: intermediate
difficulty: ⭐⭐
```

**Advanced Techniques:**
- **Question Prioritization:** Tackle easy questions first
- **Time Allocation:** Budget time per section
- **Pattern Recognition:** Identify common question types
- **Confidence Scoring:** Mark uncertain answers for review

**Strategy Matrix:**
```
Question Type    | Time Limit | Strategy
-----------------|------------|------------------
Multiple Choice  | 1 min      | Eliminate → Select
True/False       | 30 sec     | Gut check first
Fill in Blank    | 2 min      | Context clues
Essay/Practical  | 5 min      | Outline → Write
```

---

### 2. Error Analysis System (错题分析系统)

```yaml
skill: error_analysis
level: intermediate
difficulty: ⭐⭐
```

**Error Categories:**
- **Knowledge Gap:** Don't know the concept
- **Careless Mistake:** Knew it but missed detail
- **Time Pressure:** Ran out of time
- **Misunderstanding:** Interpreted question wrong

**Analysis Template:**
```yaml
error_log:
  - question_id: "Q12345"
    error_type: "knowledge_gap"
    topic: "water_quality"
    correct_answer: "A"
    my_answer: "C"
    reason: "Didn't know ammonia affects pH"
    action: "Study water chemistry chapter"
    review_date: "2026-03-14"
```

---

### 3. Peer Learning Network (同伴学习网络)

```yaml
skill: peer_learning
level: intermediate
difficulty: ⭐⭐
```

**Building Your Network:**
1. Find 3-5 study partners at similar level
2. Create shared study schedule
3. Form mini study group
4. Peer review each other's understanding

**Collaboration Techniques:**
- **Study Sessions:** Regular group reviews
- **Teaching Each Other:** Explain concepts to peers
- **Accountability Partners:** Check on each other's progress
- **Resource Sharing:** Share helpful materials

---

### 4. Mode Mastery (模式精通)

```yaml
skill: mode_mastery
level: intermediate
difficulty: ⭐⭐
```

**Deep Dive into Your Chosen Mode:**

#### Cultivation Mode Mastery
- Learn all 7 realm progression requirements
- Understand cultivation techniques
- Master meditation for focus
- Study ancient texts (exam archives)

#### Study Mode Mastery
- Develop systematic learning approach
- Create effective note-taking system
- Build spaced repetition schedule
- Balance breadth vs depth

#### Parenting Mode Mastery
- Design engaging learning activities
- Balance play and education
- Track child's learning preferences
- Celebrate small wins effectively

#### Hell Mode Mastery
- Build mental resilience
- Develop stress management
- Optimize under pressure
- Learn from failures quickly
