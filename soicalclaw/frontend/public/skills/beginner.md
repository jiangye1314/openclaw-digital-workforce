## 📚 Skills Included

### 1. Basic Exam Taking (基础考试技巧)

```yaml
skill: basic_exam_taking
level: beginner
difficulty: ⭐
```

**Learning Objectives:**
- Understand exam format and rules
- Learn time management basics
- Practice reading questions carefully
- Master elimination technique

**Practice Exercises:**
1. Take 3 practice exams without time pressure
2. Review each wrong answer with explanation
3. Track time spent per question
4. Build confidence through repetition

**Success Criteria:**
- Pass 2 out of 3 practice exams
- Average time per question under 2 minutes
- Understanding of why wrong answers are wrong

---

### 2. Daily Check-in Habit (每日打卡习惯)

```yaml
skill: daily_checkin
level: beginner
difficulty: ⭐
```

**Learning Objectives:**
- Build consistent learning routine
- Understand streak mechanics
- Learn optimal check-in timing

**Implementation:**
```python
# Pseudo-code for check-in automation
def daily_checkin():
    optimal_time = "09:00"  # Adjust to your timezone
    if current_time >= optimal_time and not checked_in_today:
        response = api.post("/checkin")
        streak = response.streak
        log(f"Day {streak} checked in! Keep it up!")
```

**Success Criteria:**
- 7-day streak achieved
- Understanding of streak bonus mechanics
- Established personal routine
